# Cisco Packet Tracer – Large Network Design Project

## About This Project
This is a large-scale network design I built for my Networking class (NETI 104) at Ivy Tech Community College. The project demonstrates my ability to design, configure, and document a complex network topology.

## Network Overview
- **Total Devices:** 437 PCs (PC0 – PC436)
- **Routers:** 3
- **Switches:** 6
- **Subnets:** 3

## IP Addressing Scheme
| Subnet | IP Range | Subnet Mask | Default Gateway |
| :--- | :--- | :--- | :--- |
| **Subnet A** | 192.168.1.0 – 192.168.1.255 | 255.255.255.0 | 192.168.1.1 |
| **Subnet B** | 192.168.2.0 – 192.168.2.255 | 255.255.255.0 | 192.168.2.1 |
| **Subnet C** | 192.168.3.0 – 192.168.3.255 | 255.255.255.0 | 192.168.3.1 |

## What I Learned
- Configured static and dynamic routing (RIP/OSPF)
- Subnetting and IP address planning for large networks
- Switch and router configuration (VLANs, trunking)
- Network testing with ping and traceroute
- Troubleshooting connectivity issues

## How to Open This File
1. Download Cisco Packet Tracer (free from Cisco Networking Academy)
2. Open the `.pkt` file in Packet Tracer
3. Explore the network topology
